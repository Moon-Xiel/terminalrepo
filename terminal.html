Les bases

Commande Résultat
Ctrl Alt T Lancer le Terminal
exit Fermer le Terminal
clear Nettoyer le Terminal
TAB Auto-complétion
!! Répète la dernière commande
Ctrl C Annuler/Abandonner
sudo apt-get install [nom] Installer
↑ Commande précédente
↓ Commande suivante

Les permissions

Commande Résultat
chmod Changer les permissions
chmod 755 Toutes les permissions
User/Group/All Ordre
  • R (lire)
  • W (modifier)
  • X (execute)
 Types

Suppression

Commande Résultat
rm -rf [nom] Supprimer dossier plein
rmdir [nom] Supprimer dossier vide
  • rm -f(force)
  • rm -r(recursive)
  • rm -i(confirmation)

Création

Commande Résultat
mkdir [nom] Créer un dossier
touch [nom] Créer un fichier
mv [ancien chemin][nouveau chemin] Déplacer
mv [ancien nom] [nouveau nom] Renommer

Types de dossiers

Commande Résultat
. Répertoire courant
.. Répertoire parent
~ HOME

Déplacements

Commande Résultat
cd [nom] Changer de répertoire
cd ou cd ~ Retourner sur HOME
cd - Retourner dans le dossier précédent
pwd Savoir son emplacement

Affichages des dossiers/fichers

Commande Résultats
ls Contenu du dossier
ls -a Contenu du dossier + fichiers cachés
ls -l Contenu du dossier + permissions
ls -la Contenu du dossier + fichiers cachés + permissions
